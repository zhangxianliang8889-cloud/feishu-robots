# 飞书群智能助手 - 功能介绍

## 🤖 我的身份定位

我是**飞书群智能助手**，专注于为企业微信群提供自动化运营支持。我的核心使命是：

> **让群聊数据有价值，让团队协作更高效**

---

## 📊 我的两大核心能力

### 1️⃣ 群会议纪要机器人

**定位**：智能内容总结专家

**核心功能**：
- 📅 **日报**：每日21:00发送，总结最近24小时群聊精华
- 📊 **周报**：每周一10:00发送，总结上周群聊重点
- 📈 **月报**：每月1日14:00发送，总结上月群聊成果

**输出内容**：
```
📌 核心话题提取
✅ 重要决议与成果
📋 待跟进事项
💡 建议与展望
🌟 金句激励
```

**技术亮点**：
- AI智能总结（豆包大模型）
- 关键词智能匹配金句
- 多维度内容分析

---

### 2️⃣ 群消息统计机器人

**定位**：数据驱动运营专家

**核心功能**：
- 📅 **日报**：每日09:00发送，统计昨日消息数据
- 📊 **周报**：每周五12:00发送，统计上周消息数据
- 📈 **月报**：每月1日15:00发送，统计上月消息数据

**输出内容**：
```
📈 消息总量与活跃率
🏆 活跃之星排行榜
📋 全员消息榜单
🌟 群氛围评估
💌 沉默用户提醒
```

**技术亮点**：
- CEO用户自动过滤
- 智能活跃度分析
- 温馨互动提示

## 🔧 我的技术架构

### 核心能力矩阵

| 能力 | 实现方式 | 状态 |
|-----|---------|------|
| 幂等执行 | TaskStateManager + 任务唯一ID | ✅ 已集成 |
| 分布式锁 | DistributedLock + 文件锁 | ✅ 已集成 |
| API限流 | RateLimiter + 滑动窗口 | ✅ 已集成 |
| 并发处理 | ThreadPoolExecutor | ✅ 已集成 |
| AI总结 | 豆包大模型API | ✅ 已集成 |
| 状态追踪 | JSON持久化存储 | ✅ 已集成 |

## 🔍 遇到问题时的自我修正指南

### 问题：发送了重复报告
**检查点**：
1. 幂等检查是否生效？
2. 任务ID是否唯一？
3. 状态文件是否正常保存？

### 问题：报告内容错误
**检查点**：
1. 数据获取是否正确？
2. AI总结是否成功？
3. 格式验证是否通过？

### 问题：发送失败
**检查点**：
1. Token是否有效？
2. 群ID是否正确？
3. 网络是否正常？

### 问题：用户名显示异常
**检查点**：
1. 用户缓存是否更新？
2. 用户是否在群内？
3. API返回是否完整？
